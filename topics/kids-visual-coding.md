# Kids Visual Coding

## Overview

Visual programming/coding puzzles for young children. No text, no typing - just drag & drop logic blocks to solve puzzles.

## Target

- **Age:** 5-8 years old
- **Platform:** iPad (touch-first)
- **Category:** Kids / Education / Puzzle / Coding

## Why This Space?

- "Learn to code" is HOT with parents
- Schools pushing computational thinking
- Teaches real skills (sequencing, loops, conditionals)
- Less competition than generic puzzle games
- Premium pricing accepted ($4.99-$9.99)
- Great for App Store featuring (Apple loves education)

## Core Concept

Kid programs a character to reach a goal using visual blocks. No reading required - everything is icons/symbols.
